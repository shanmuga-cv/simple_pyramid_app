simple_pyramid_app README
==================
A minimal example to showcase a authentication and authorization in pyramid framework.

This mimics a student signup.
* Sign up page is not accessible after login
* Student list can be shown to only logged in users

Getting Started
---------------

- cd <directory containing this file>

- $VENV/bin/python setup.py develop

- $VENV/bin/initialize_simple_pyramid_app_db development.ini

- $VENV/bin/pserve development.ini
